Logistic Regression Model

Pre-processing steps for each model:

  • Split data into X = data & y = diagnosis
  • Label encoded the y data to change from string (B for benign, M for malignant) to int (0 for benign, 1 for malignant)
  • Scaled the X data using StandardScaler
  • Split the datasets into training & testing using train_test_split with random_state = 42
  • From sklearn.svm import SVC (kernel = “rbf”)
  • Fit the training data to the model
  • Testing Data Score: 0.986
  • Made predictions for y values
  • Created confusion matrix to compare predictions to actual
  • Confusion matrix shows that the model did not overpredict malignancy, and it did well on a low number of False Negatives (2 predicted benign but actually malignant)
